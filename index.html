Robomakery

Open source robots for people with mobility impairments

Current Projects

Spot

Spot is a small wheeled robot that is designed to help people pick up small objects from the floor (like a remote control or phone).

Learn more: https://github.com/robomakery/spot

Walkchair

Walkchair is a project exploring how to add legs to power wheelchairs for increased mobility.

Learn more: http://www.walkchair.org
